## Tasks ES6

### 1. Default Parameter

Buat function bernama **greet** dengan default parameter `name = "Guest"`.

Contoh output:

- `greet()` → `"Hello, Guest!"`
- `greet("Dino")` → `"Hello, Dino!"`

---

### 2. Arrow Function

Buat arrow function bernama **square** yang menerima satu angka dan mengembalikan kuadrat angka tersebut.

Contoh:

- `square(6)` → `36`

---

### 3. Template Literal

Diberikan variabel berikut:

```js
const brand = "Sadino Tech";
const service = "Web Development";
```

Buat satu kalimat dengan template literal:  
`"Sadino Tech menyediakan layanan Web Development."`

---

### 4. Object Destructuring

Dari object berikut:

```js
const course = {
  title: "JavaScript Dasar",
  mentor: "Andi",
  duration: 5,
};
```

Gunakan **destructuring** untuk mengambil `title` dan `duration`, lalu tampilkan:  
`JavaScript Dasar - 5 hari`

---

### 5. Class ES6 (Inheritance)

Buat class **Vehicle** dengan:

- properties: `brand`, `year`
- method: `getInfo()` → `"Brand: <brand>, Year: <year>"`

Lalu buat class **Car** yang extends Vehicle, dengan:

- property tambahan: `type`
- method: `getCarInfo()` → `"Brand: <brand>, Type: <type>, Year: <year>"`

Contoh penggunaan:

```js
const car1 = new Car("Toyota", 2020, "SUV");
car1.getCarInfo();
// Brand: Toyota, Type: SUV, Year: 2020
```

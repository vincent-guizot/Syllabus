## **Tasks â€” ARRAY**

### **1. Jumlah Elemen Array (Easy)**

Hitung **total semua elemen** dari array berikut:

```js
const numbers = [4, 7, 2, 9, 1];
```

---

### **2. Nilai Terbesar (Easy)**

Temukan **nilai terbesar** dari array berikut, tanpa menggunakan `Math.max()`:

```js
const values = [12, 5, 20, 7, 3];
```

---

### **3. Menampilkan Angka Genap (Easy)**

Tampilkan **hanya elemen genap** dari array berikut:

```js
const data = [10, 13, 8, 7, 6, 20];
```

---

### **4. Hapus Nilai Duplikat (Medium)**

Buat array baru yang **tanpa duplikat**, tidak boleh menggunakan `new Set()`:

```js
const items = [2, 4, 4, 7, 2, 9, 9, 1];
```

---

### **5. Rotasi Array ke Kanan (Hard)**

Buat fungsi untuk merotasi array ke kanan sebanyak **n kali**.

Contoh:

```js
const arr = [1, 2, 3, 4, 5];
const n = 2;
```

Hasil yang diharapkan:

```
[4, 5, 1, 2, 3]
```

> Rotasi ke kanan berarti elemen paling belakang berpindah ke depan.

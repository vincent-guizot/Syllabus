```js

```

### **4. Medium â€” Fungsi Mencari Nilai Terbesar dalam Array**

Buat function `findMax` yang menerima sebuah array angka dan mengembalikan nilai terbesar di dalamnya.  
_Tidak boleh menggunakan `Math.max()`._

Contoh:

```js
findMax([2, 9, 3, 7]); // Output: 9
```

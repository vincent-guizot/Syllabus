## Cara Mengerjakan

1. Clone repository

   ```
   git clone <link repository>
   ```

2. Buat branch masing - masing

   ```
   git checkout -b namabranch
   ```

3. Kerjakan dalam branch masing - masing

4. Add, Commit, Push

5. Push ke dalam branch masing - masing

    ```
    git push origin namabranch
    ```

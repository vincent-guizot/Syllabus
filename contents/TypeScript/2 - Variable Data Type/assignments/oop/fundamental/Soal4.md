# Exercise 4 — Movie Rating Checker

## Summary

Pada latihan ini peserta akan menggabungkan class dengan conditional logic.

Peserta belajar bahwa method tidak hanya menampilkan data, tetapi juga dapat menjalankan logika.

---

## Outline

Materi yang digunakan:

- Class
- Method Logic
- Conditional Statement
- Boolean Decision

---

## Tasks

Buat class bernama `Movie`.

### Properties

- title → string
- genre → string
- rating → number

---

### Method

Buat method:

`isRecommended()`

Rules:

- Jika rating >= 8 → tampilkan "Recommended"
- Jika rating < 8 → tampilkan "Not Recommended"

---

### Create Object

Buat minimal 3 movie.

---

### Output

Tampilkan:

- Judul
- Rating
- Status rekomendasi

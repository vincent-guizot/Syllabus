# Exercise 3 — Car Information System

## Summary

Latihan ini memperkenalkan penggunaan optional property pada TypeScript.

Peserta akan belajar membuat object yang memiliki data wajib dan data opsional.

---

## Outline

Materi yang digunakan:

- Optional Property
- Constructor
- Class Structure
- Object Modeling

---

## Tasks

Buat class bernama `Car`.

### Properties

- brand → string
- model → string
- year → number
- color → optional string

---

### Method

Buat method:

`showCarInfo()`

Method menampilkan seluruh informasi mobil.

---

### Create Object

Buat minimal 2 object mobil:

- 1 mobil dengan warna
- 1 mobil tanpa warna

---

### Output

Tampilkan seluruh data mobil.

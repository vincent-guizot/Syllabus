# Exercise 1 — Student Biodata System

## Summary

Pada latihan ini, peserta akan belajar membuat class sederhana menggunakan TypeScript.

Peserta akan memahami bagaimana cara mendefinisikan property, constructor, dan method untuk merepresentasikan data seorang student.

Latihan ini bertujuan memperkenalkan dasar Object Oriented Programming secara sederhana.

---

## Outline

Materi yang digunakan:

- Type Annotation
- Class
- Constructor
- Object Instance
- Method

---

## Tasks

Buat sebuah class bernama `Student`.

### Properties

- name → string
- age → number
- major → string

---

### Method

Buat method:

`showProfile()`

Method ini harus menampilkan:

- Nama
- Umur
- Jurusan

---

### Create Object

Buat 2 object student berbeda.

Contoh:

- Budi
- Siti

---

### Output

Tampilkan profile dari masing-masing student.

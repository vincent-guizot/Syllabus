# Exercise 2 — Product Catalog

## Summary

Pada latihan ini, peserta akan membuat sistem katalog produk sederhana.

Peserta belajar bagaimana object dapat digunakan untuk merepresentasikan data produk dan menampilkan informasi detail melalui method.

---

## Outline

Materi yang digunakan:

- Class
- Constructor
- Number Type
- Method
- Object Creation

---

## Tasks

Buat class bernama `Product`.

### Properties

- name → string
- price → number
- stock → number

---

### Method

Buat method:

`showDetail()`

Method menampilkan:

- Nama Produk
- Harga
- Stok

---

### Create Object

Buat minimal 3 object produk.

Contoh:

- Laptop
- Mouse
- Keyboard

---

### Output

Tampilkan seluruh detail produk.

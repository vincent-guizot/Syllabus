# Exercise 2 — Payment Interface

## Summary

Pada latihan ini peserta akan memahami interface sebagai contract dalam TypeScript.

Peserta belajar bahwa class dapat diwajibkan memiliki method tertentu.

---

## Outline

Materi yang digunakan:

- Interface
- implements
- Method Contract

---

## Tasks

Buat interface bernama `Payable`.

Method:

`pay(amount: number): void`

---

Buat 2 class:

- CashPayment
- EWalletPayment

Kedua class wajib mengimplementasikan interface.

---

### Behavior

CashPayment:

Menampilkan pembayaran tunai.

EWalletPayment:

Menampilkan pembayaran digital.

---

## Create Object

Buat object dari kedua class.

---

## Output

Simulasikan pembayaran.
